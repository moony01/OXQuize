# OX Quiz (Java)

> Console-based **true/false (OX) quiz game** built in Java as a learning project.

[![Java](https://img.shields.io/badge/Java-orange?logo=java)](https://www.java.com/)

---

## Overview

A simple console application that runs a true/false quiz game. Built early in my Java journey to practice control flow, file I/O, and basic OOP patterns.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Language** | Java |
| **Runtime** | JVM |

## Run

```bash
git clone https://github.com/moony01/OXQuize.git
cd OXQuize

# Compile and run from your IDE (Eclipse / IntelliJ)
# or via the command line:
javac OXQ/*.java
java OXQ.Main
```
